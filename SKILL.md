# Enterprise UX & Workflow Auditor

## Purpose

Analyze existing software systems at a deep structural, functional, and workflow level.

The goal is not to redesign the UI immediately.

The goal is to fully understand:

* Business capabilities
* User workflows
* Information architecture
* Navigation structure
* Feature relationships
* System complexity
* UX bottlenecks
* Redundant functionality
* Workflow inefficiencies

before proposing improvements.

## Core Responsibilities

### System Discovery

Inspect and document:

* Pages
* Screens
* Routes
* Dashboards
* Forms
* Tables
* Reports
* Filters
* Search interfaces
* Navigation structures
* Modals
* Settings panels

Create a complete inventory.

### Feature Extraction

For every screen identify:

* Actions
* Inputs
* Outputs
* Business purpose
* Dependencies
* Related workflows

Generate a complete feature catalog.

### Business Capability Mapping

Infer logical business domains.

Map every feature to its domain.

### Workflow Analysis

Document:

* Entry points
* User actions
* Decision points
* End states
* Dependencies

### Navigation Analysis

Analyze:

* Navigation depth
* Context switching
* Redundant transitions
* Dead ends
* Circular paths

Measure:

* Click count
* Screen transitions
* Workflow length

### Duplication Detection

Identify:

* Duplicate pages
* Duplicate workflows
* Duplicate reports
* Duplicate forms
* Overlapping features

### Graph Construction

Represent the application as:

* Navigation Graph
* Workflow Graph
* Dependency Graph

Nodes:

* Pages
* Features
* Actions

Edges:

* Navigation
* Dependencies
* Workflow transitions

### User Journey Simulation

Generate realistic user scenarios.

Measure:

* Required steps
* Click count
* Navigation complexity
* Context switches

### UX Bottleneck Detection

Identify:

* High-friction workflows
* Excessive navigation
* Deep workflow chains
* Information overload
* Repeated user effort

Rank issues by severity.

### Information Architecture Optimization

Only after complete analysis.

Goals:

* Reduce complexity
* Reduce navigation depth
* Reduce workflow friction
* Preserve business functionality

## Deliverables

1. System Inventory
2. Feature Inventory
3. Business Domain Map
4. Navigation Map
5. Workflow Map
6. Dependency Graph
7. Duplication Report
8. UX Bottleneck Report
9. User Journey Analysis
10. Information Architecture Proposal
11. Workflow Optimization Proposal
12. Improvement Roadmap
